- ## Installation
	- `brew install fzf` #macOS
- ## To install useful key bindings and fuzzy completion:
	- `$(brew --prefix)/opt/fzf/install` #macOS
- ## CLI History FZF
	- Press `ctrl + r`
- ## FZF With Preview
	- #+BEGIN_NOTE
	  You should install [[prgmentize]] first
	  #+END_NOTE
	- `fzf --preview 'pygmentize -g {}'`
